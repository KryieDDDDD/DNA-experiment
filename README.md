To run the programme to decode oligos, please download the DNA data from https://doi.org/10.6084/m9.figshare.31677721.
To decode scheme1, call function "fountain_dec_from_txt()" in main.cpp and set n = 28500 in 'DNA_system_parameter.txt'.
To decode scheme2, call function "SPCC_N_foutain_dec_from_txt()" in main.cpp and set n = 28970 in 'DNA_system_parameter.txt'.
