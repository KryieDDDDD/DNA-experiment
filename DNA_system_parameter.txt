The lines beginning with '#' are used to set parameter, which have the form: #par { value(s) }. 
1) par cannot contain blank charachters;
2) between par and {, there must be at least one blank, and non-blank charachters are not allowed;
3) other words except those between '#' and '}' will not be interpreted.


#outer_code    { R }R for Raptor code

#inner_code	{ P }	R, L, S, P for RS, Levenshtein (2logN), SVT (logN+logP), single parity check code(2nt) scheme, respectively

#N	{ 248 }	number of bases in each oligo

#P	{ 16 }	for the SVT code; at least be the maximum run-length of bit 0, 2, ... of an oligo; effective only if SVT code is used

#deg	{ 8 } 	for RS code in GF(2^deg)
#dmin	{ 3 }	the minimum Hamming distance of RS code, current decoding strategy only works for dmin = 3, i.e., two redundant symbols

#use_BP	{ 0 }	LT decoding: 1: BP; 0: SGE/inactivation decoding

#RSD_delta	{ 0.001 }	delta for RSD
#RSD_c		{ 0.025 }	c for RSD 

#n		{ 28500 }		number of source symbols of outer LT codes 28500/28970-1primer/28724-4primers
#write_size	{ 30000 }		number of writing/systhesized oligos (encoded symbols of outer LT codes)
#read_size	{ 1200000 }	number of reading/sequencing oligos

#bw_seed	{ 16 }		bit-with of seed for each encoded symbol (log(write_size) <= bw_seed <= 32)

#c_gc		{ 0.05 }	GC content belongs to [0.5 - c_gc, 0.5 + c_gc]
#max_run	{ 3 }		maximum run-length for oligo
#check_GC_run	{ 1 }		1 for checking GC-content and run-length; 0 for not

#detect_only	{ 0 }	1 for detection only and 0 for error correction

#pi { 0.001 }	inseration error probability
#pd { 0.004 }	deletion error probability
#ps { 0.005 }	substitution error probability

#ATCG_to_bits { 00 01 10 11 }	the mapping between ATCG and bits (most to least significant)

The transition probability among ATCG conditioned on that a substitution error has happened, where ATCG respectively correspond to rows from top to bottom and columns from left to right. 
E.g., the number at the first row and second column indicates the transition probability from A to T.
Each row will be automatically normalized.
